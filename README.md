## Sobre o Projeto

DevFinances é uma aplicação simples de controle financeiro pessoal, desenvolvido durante as aulas da [Maratona Discover](https://www.youtube.com/playlist?list=PLeLKux5eT3kY2mvZUi7IM5T548vfKxGq5).
